https://ebbnflow.tistory.com/196 (사용 방법 알려주는 사이트)




1, 아무데나 폴더생성

2, 마우스 오른쪽 클릭 후 GIT BASH HERE 클릭

3, 초기화 (초기 세팅)
git init

git config --global user.name "본인의깃닉네임" 

git config --global user.email "본인의깃이메일"

4, 명령어

git status 
-현재 상태를 확인하는 명령어


5, 변경사항 더하기 : git add
-커밋할(깃에 반영할) 내용들을 추가할 수 있고 이를 스테이징 이라고한다.

git add -A
git add --all
git add .
git add *
-모든 변경사항 반영할때 사용

6, 변경사항 반영하기 : git commit
git commit -m "원하는 메세지"
-변경사항 반영하기  커밋 한 내용을 깃이 추적하여 버전관리를 할 수 있도록 도와줌 24시간 내 추적

7, 반영사항 확인하기 : git log

git log
-방금 전 커밋한 내용확인 노란 줄 commit 뒤에 나온 번호는 상태의 일련번호이고 이번호를 통해 분기할 수 있음

8-1, 이전 상태로 되돌리기 : git reset
git reset 커밋번호6자리 --hard
-커밋번호는 git log 로 노란 줄 commit 뒤에 나온 앞번호부터 6자리이다. 한번 과거 파일로가면 전파일로 되돌리지못함

8-2, 이전 상태로 되돌리기2 : git revert
git revert 커밋번호여섯자리
-이전 상태로 돌림 과거 파일로 가도 전파일로 되돌릴 수 있음
